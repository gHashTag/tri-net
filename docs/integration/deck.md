# TRI-NET -> Сеть 2030 / суверенная АСУ ТП -- дек для ТРГ (6 слайдов)

Handoff-ready outline. Full pitch: docs/pitch/tri-net-asutp... see the two live
artifacts (pitch + integration brief). This is the government-working-group deck.

## Слайд 1 -- Проблема и мандат
- Президентская задача: +порядок автоматизации; суверенная открытая АСУ ТП.
- МСЭ-Т SG-13 «Сети 2030»: слияние вычислительной и телеком-парадигм.
- Письмо ПАО «Ростелеком» (расп. 3339-Р): Балыбердин -- эксперт ТРГ по шине OCF.
- Тезис: конвергенция «вычисления + связь» нужна НА ОДНОМ чипе, суверенно, открыто.

## Слайд 2 -- Наш ответ: один примитив, один чип
- Троичный sign-select MAC {-1,0,+1} = 0 DSP; тот же примитив -- радио И ИИ.
- Радио (AD9361) + обученный ИИ на одном Zynq-7020, вживую -- конвергенция на кремнии.
- Троичность {-1,0,+1} = его семантика {0,1,нет-данных}: «0» -- и пропуск в MAC,
  и firing-gate функционального блока МЭК-61499.

## Слайд 3 -- Железные доказательства (за 2 недели)
- Читаемый текст по эфиру 2.4 ГГц, BER=0/576 бит; CDMA два узла в одной полосе, BER=0.
- Обученная 5-классовая модель сети на ARM узла (99.8%); CSMA-MAC (сенс->передача) вживую.
- Троичный код-LM генерирует код на самом узле.
- Открытый флоу без Vivado -> загружаемый PS7-битстрим 4.05 МБ @ 308 МГц.

## Слайд 4 -- Прямая приёмка АСУ ТП (AC-2)
- Adaptive clock recovery (его FIFO+ФАПЧ = МСЭ-Т G.8261) в RTL: NCO+FIFO+PI.
- iverilog: идеальный захват, ошибка 0.000000%, буфер на 50%.
- Пост-P&R на xc7z020 = 122 МГц -- детерминированная петля восстановления такта в железе.
- Функциональный блок МЭК-61499 на троичном примитиве: firing-gate {нет-данных} verified.

## Слайд 5 -- Разделение труда (комплементарно, не конкуренция)
- Мы: ALU/радио/ИИ + открытый флоу + воспроизводимость (φ-anchor/SHA) + RTL-петли.
- Он: сеть ССИ (inter-PE fabric) + polyhedral-математика «большого цикла» (5+ лет).
- Точка стыка: общий UNI/NNI по ИК-13; его ССИ внутри slice, наш TRI MESH между узлами.

## Слайд 6 -- Дорожная карта и просьба
- Artix-7/Zynq-7020 (отработка, есть) -> Pango Logos-2 / P100 (RU-порт) -> кремний TRI-1.
- Смета MVP на 3 узла ~218 тыс ₽ (есть).
- Просьба: pilot-объект АСУ ТП с SLA (jitter/latency budget, целевая частота АЦП/ЦАП);
  обновлённый текст ЯПФ-компилятора -- дадим честную техническую рецензию.
